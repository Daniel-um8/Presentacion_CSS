Box Model

El Box Model de CSS es un concepto fundamental que define cómo se estructura y se representa un elemento HTML en una página web. Cada elemento se puede visualizar como una caja rectangular, y el Box Model describe las diferentes partes de esta caja.

Las partes del Box Model son:

  • Contenido (Content): El área donde se muestra el texto o las imágenes del elemento.

  • Relleno (Padding): El espacio entre el contenido del elemento y su borde. Se puede ajustar para proporcionar espacio interior adicional.

  • Borde (Border): Una línea que rodea el padding (si se ha definido). El borde puede tener diferentes estilos, colores y grosores.

  • Margen (Margin): El espacio fuera del borde, separando el elemento de otros elementos adyacentes. Aumenta el espacio entre el elemento y sus alrededores.
