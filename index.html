¿Qué es CSS?

(Cascading Style Sheets) es un lenguaje de diseño utilizado para describir la presentación de un documento escrito en HTML. CSS se encarga de controlar el aspecto visual de una página web, como los colores, fuentes, márgenes, espaciado, tamaños, alineaciones y el diseño en general.
